# Prompt Change Pull Request

## Ziel
Beschreibe kurz, was verbessert werden soll.

## Änderungen
- Welche Prompt-Dateien oder Regeln wurden geändert?
- Wurden neue Testfälle ergänzt?
- Wurden Code oder Evaluationsskripte angepasst?

## Warum
Welches konkrete Problem soll gelöst werden?

## Getestet mit
- Welche Beispielinputs oder Use Cases wurden betrachtet?
- Welche Beobachtungen gab es?

## Review-Fokus
- Ist die Aufgabe klarer als vorher?
- Ist das Ausgabeformat stabiler?
- Sind Regeln konsistent und nicht widersprüchlich?
- Ist die Änderung klein und nachvollziehbar?
